# 把个人博客做得更像作品集

我不太想把博客做成纯模板站。即使内容是算法、建模和技术笔记，页面也应该有一点辨识度，而不是任何人换个标题都一样。

## 我在意的不是花哨，而是统一

真正影响观感的通常不是炫技，而是三件事：

- 首页、列表页、详情页是不是同一种语气
- 色彩和留白是不是稳定
- 链接、按钮、卡片这些重复元素是不是形成了统一语言

## 为什么选择 VitePress

因为它允许我把内容维护放在第一位。Markdown 负责写作，主题样式负责视觉，必要时再用 Vue 做扩展，这样不会一开始就把项目做重。

## 这次改版的原则

1. 去掉无效链接和模板占位内容。
2. 统一背景和卡片系统。
3. 保留足够的后续扩展空间。

后面如果继续改，我更可能做的是内容驱动和结构化，而不是堆越来越复杂的视觉特效。
