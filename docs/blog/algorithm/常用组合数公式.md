## ICPC 常用组合数公式与技巧
- ### 1. **基本公式**
- **组合数定义**：
	  \[ C_n^k = \binom{n}{k} = \frac{n!}{k!(n-k)!} \]
	  **递推关系（杨辉三角）**：
	  \[ \binom{n}{k} = \binom{n-1}{k} + \binom{n-1}{k-1} \]
	  **对称性**：
	  \[ \binom{n}{k} = \binom{n}{n-k} \]
- ### 2. **经典恒等式**
- **二项式定理**：
	  \[ (a + b)^n = \sum_{k=0}^{n} \binom{n}{k} a^{k} b^{n-k} \]
	  **列求和**：
	  \[ \sum_{k=r}^{n} \binom{k}{r} = \binom{n+1}{r+1} \]
	  **行求和**：
	  \[ \sum_{k=0}^{n} \binom{n}{k} = 2^n \]
	  **变号求和**：
	  \[ \sum_{k=0}^{n} (-1)^k \binom{n}{k} = 0 \quad (n \geq 1) \]
	  **平方和**：
	  \[ \sum_{k=0}^{n} \binom{n}{k}^2 = \binom{2n}{n} \]
	  **范德蒙德卷积**：
	  \[ \sum_{k} \binom{r}{k} \binom{s}{n-k} = \binom{r+s}{n} \] ，其中$max⁡(0,n−s)≤k≤min⁡(r,n)$
  - **完整公式**：
		  \[
		  \boxed{\sum_{k=0}^{n} \binom{r}{k} \binom{s}{n-k} = \binom{r+s}{n} }
		  \]
  - 1. **组合意义**：
  - 左侧：从两个独立集合中选**总计 \(n\) 个元素**的所有可能方式。
    - 第一个集合有 \(r\) 个元素，选 \(k\) 个（共 \(\binom{r}{k}\) 种选法）。
    - 第二个集合有 \(s\) 个元素，选 \(n-k\) 个（共 \(\binom{s}{n-k}\) 种选法）。
  - 2. **约束条件**：
  - 右侧：直接从合并后的集合（共 \(r+s\) 个元素）中选 \(n\) 个元素（共 \(\binom{r+s}{n}\) 种选法）。
  - \(r, s \in \mathbb{N}^0\)（非负整数），\(n \in \mathbb{Z}\)（整数）。
  - 当 \(k > r\) 或 \(n-k > s\) 时，\(\binom{r}{k} = 0\) 或 \(\binom{s}{n-k} = 0\)，因此求和范围 **隐式限定** 在：\[\max(0, n-s) \leq k \leq \min(r, n)\]
- ### 3. **特殊序列**
- **卡特兰数**：
	  \[ C_n = \frac{1}{n+1} \binom{2n}{n} = \binom{2n}{n} - \binom{2n}{n+1} \]
	  应用：括号序列、二叉树计数、凸多边形三角剖分。
- **错位排列数（Derangement）**：
	  \[ D_n = n! \sum_{k=0}^{n} \frac{(-1)^k}{k!} \]
	  递推式：\( D_n = (n-1)(D_{n-1} + D_{n-2}) \)，其中 \( D_0=1, D_1=0 \)
- ### 4. **组合数取模技巧**
- **模质数 \( p \)（Lucas 定理）**：
	  \[ \binom{n}{k} \equiv \binom{n/p}{k/p} \cdot \binom{n \bmod p}{k \bmod p} \pmod{p} \]
	  代码实现：
	  ```cpp
	  const int p = 1000000007;
	  int C(int n, int k) {
	    if (k  n) return 0;
	    return fac[n] * inv[k] % p * inv[n-k] % p;
	  }
	  int lucas(int n, int k) {
	    if (k == 0) return 1;
	    return C(n%p, k%p) * lucas(n/p, k/p) % p;
	  }
	  ```
	  
	  **模非质数**：
- 扩展 Lucas 定理
- 中国剩余定理（CRT）分解
- ### 5. **生成函数**
- **普通生成函数（OGF）**：
	  如背包计数：
	  \[ \prod_{i}(1 + x^{w_i}) \longleftrightarrow \sum_{k} \text{重量 } k \text{ 的方案数} \]
	  **指数生成函数（EGF）**：
	  如排列计数：
	  \[ e^x = \sum_{n} \frac{x^n}{n!} \quad \longrightarrow \quad \text{排列数生成函数} \]
- ### 6. **容斥原理**
- **基本形式**：
  \[ \left| \bigcup_{i=1}^{n} A_i \right| = \sum |A_i| - \sum |A_i \cap A_j| + \cdots + (-1)^{n+1} |A_1 \cap \cdots \cap A_n| \]
  **错排问题**：
  \[ D_n = n! \sum_{k=0}^{n} \frac{(-1)^k}{k!} \]
- ### 7. **多项式系数**
- **定义**：
	  \[ \binom{n}{k_1, k_2, \dots, k_m} = \frac{n!}{k_1! k_2! \cdots k_m!} \quad (k_1 + \cdots + k_m = n) \]
	  **多项式定理**：
	  \[ (x_1 + \cdots + x_m)^n = \sum \binom{n}{k_1, \dots, k_m} \prod x_i^{k_i} \]
- ### 8. **路径计数**
- **格点路径数**：
	  从 \((0,0)\) 到 \((n,m)\) 且只能向右/上走：
	  \[ \text{总路径数} = \binom{n+m}{n} \]
	  **避免障碍**：
	  使用容斥或 DP 计数绕开障碍。
	  **格点路径数**：
	  从 \((0,0)\) 到 \((n,m)\) 且只能向右/上走：
	  \[ \text{总路径数} = \binom{n+m}{n} \]
	  **避免障碍**：
	  使用容斥或 DP 计数绕开障碍。
