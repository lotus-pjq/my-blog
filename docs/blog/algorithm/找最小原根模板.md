- 如果一个数有原根，可以先求出最小正原根。
- >王元于 1959 年证明了若 *m* 有原根，其最小原根是不多于 *m*0.25 级别的。
  事实上，由大量试验数据可以发现，对于足够大的 *m*，其最小正原根的大小不是多项式级别的。
  ——百度百科
