## 树上任意两节点之间最长的简单路径即为树的直径。一棵树可以有多条直径，他们的长度相等。
### 方法1.两次DFS：缺点：仅适用于正边权，优点：方便记录直径的路径，时间复杂度：O(n）
### 方法2.树形DP：优点：也适用于负边权，缺点：不方便记录直径的路径，时间复杂度：O(n)
## 求直径长度和直径路径- ### 模板1：边权为1的树的直径长度- ## B4016 树的直径
  #### 题目描述
给定一棵 $n$ 个结点的树，树没有边权。请求出树的直径是多少，即树上最长的不重复经过一个点的路径长度是多少。
  #### 输入格式
第一行输入一个正整数 $n$，表示结点个数。
第二行开始，往下一共 $n-1$ 行，每一行两个正整数 $(u,v)$，表示一条边。
  #### 输出格式
输出一行，表示树的直径是多少。
  #### 输入输出样例 1
  - >5
1 2
2 4
4 5
2 3
  ##### 输出 1
```
3
```

数据保证，$1 \leq n \leq 10^5$。
