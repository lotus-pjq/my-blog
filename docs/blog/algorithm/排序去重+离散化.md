- 涉及到三个很有用的函数: lower_bound() , upper_bound() , unique()
- ### lower_bound()和upper_bound()函数（STL）
	  二者均定义于头文件 `` 中
	  三个参数：**（起始地址，终止地址+1，目标值）**
	  **upper_bound(a.begin(),a.end(),val)**：查找第1个==大于==x的元素的位置。 
	  **lower_bound(a.begin(),a.end(),val)**：查找第1个==大于等于==x的元素的位置
- ### unique()函数（相邻的重复元素去除)
	  包含在`#include`
	  **unique(a.begin(),a.end());**：返回值为去重后数列中的end()地址（真实地址）。
	  **搭配sort实现整体数组去重(O(n+nlogn))**
	  ```C++
	  sort(a.begin(),a.end());
	  int cnt=unique(a.begin(),a.end())-a.begin();
	  ```
	  ```C++
	  sort(a+1,a+n+1);
	  //同时实现'去重'和'求去重后的元素个数'
	  int cnt=unique(a+1,a+n+1)-(a+1);
	  //cnt为去重后的数量
	  ```
- ### 不去重离散化
  ```C++
  //b[]:原数据，a[]:离散化后结果
  
  for(int i=1;i>a[i],b[i]=a[i];
  sort(b+1,b+n+1);
  for(int i=1;i>a[i],b[i]=a[i];
  sort(b+1,b+n+1);
  int cnt=unique(b+1,b+n+1)-(b+1);//去重 
  for(int i=1;i<=cnt;i++)
  	a[i]=lower_bound(b+1,b+1+n,a[i])-b;
  ```
