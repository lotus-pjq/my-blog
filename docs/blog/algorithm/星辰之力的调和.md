# 题目名称：星辰之力的调和 (Harmony of Starlight)
- ## 【题目描述】
  
  在古老的星辰大陆上，占星师小 S 负责维护天文台的**星辰共鸣阵**。共鸣阵需要精确地以频率 $n$ 运行，才能与天穹中的"恒星之眼"保持同步。
  
  天文台仓库中存有 $k$ **种**星辰碎片，第 $i$ 种碎片蕴含的星力频率为 $a_i$（每升碎片恰好提供 $a_i$ 单位的星力）。每种碎片的储量无限。
  
  小 S 需要将若干升星辰碎片投入共鸣阵的调和池中（每次投入 $1$ 升，可以选择任意一种碎片，同种碎片可以多次选取）。调和池会自动将所有碎片的星力均匀混合——也就是说，如果投入了 $m$ 升碎片，那么调和后的星力频率为：
  
  $$
  \text{频率} = \frac{\sum_{j=1}^{m} a_{s_j} }{m}
  $$
  
  其中 $s_j$ 为第 $j$ 升碎片所选择的种类。
  
  小 S 希望调和后的频率**恰好等于** $n$。由于星辰碎片十分珍贵，他想知道**最少需要投入多少升碎片**才能实现这一目标。
  
  如果无论如何都无法达到频率 $n$，输出 $-1$。
- ## 【输入格式】
  
  第一行包含两个正整数 $n$ 和 $k$，分别表示目标频率和星辰碎片的种类数。
  
  第二行包含 $k$ 个**非负**整数 $a_1, a_2, \dots, a_k$，表示每种碎片的星力频率。
- ## 【输出格式】
  
  输出一个整数，表示最少需要投入的碎片升数。若无解则输出 $-1$。
- ## 【样例数据】
- ### 样例 1
  
  **输入**
  
  ```
  3 2
  5 1
  ```
  
  **输出**
  
  ```
  2
  ```
  
  **解释**：选 $1$ 升频率 $5$ 的碎片和 $1$ 升频率 $1$ 的碎片，均值 $= (5+1)/2 = 3$，恰好等于目标频率。
  
  ---
- ### 样例 2
  
  **输入**
  
  ```
  5 3
  3 6 8
  ```
  
  **输出**
  
  ```
  3
  ```
  
  **解释**：选 $2$ 升频率 $6$ 的碎片和 $1$ 升频率 $3$ 的碎片，均值 $= (6+6+3)/3 = 5$，恰好等于目标频率。可以验证不存在仅用 $1$ 或 $2$ 升的方案。
  
  ---
- ### 样例 3
  
  **输入**
  
  ```
  7 2
  2 4
  ```
  
  **输出**
  
  ```
  -1
  ```
  
  **解释**：所有碎片频率均小于 $7$，无论如何混合均值 $\le 4 < 7$，无解。
- ## 【数据范围】
- $0 \le n \le 1000$
- $1 \le k \le 10^6$
- $0 \le a_i \le 1000$
- ## 【提示】
- 可以多次选取同一种碎片。
- 注意 $a_i$ 可以等于 $0$（空白碎片，只增加体积、不贡献星力）。
- 总投入量至少为 $1$ 升。
- ## 【题解】
  
  参见 [CF788C_题解.md](typora://app/CF788C_%E9%A2%98%E8%A7%A3.md)，核心思路完全相同：
- 令 $c_i = a_i - n$，问题等价于选若干个 $c_i$（可重复），总和恰为 $0$，最小化选取个数。
- 由于 $c_i \in [-1000, 1000]$，若解存在，中间前缀和可控制在此范围内。
- BFS 从 $0$ 出发，每步可走 $+c_i$，仅在 $[-1000, 1000]$ 范围搜索，首次回到 $0$ 即为答案。
- 复杂度 $O(2001 \times \min(k, 2001))$。
