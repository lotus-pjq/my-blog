# 线性基完全指南：从高斯消元到贪心构造

线性基是处理异或问题的强大工具，本文系统总结线性基的理论、构造方法和经典应用。

## 什么是线性基

给定 n 个整数，线性基是这些数的一个子集，满足：
1. 原集合中任意数都可以由线性基中的数异或得到
2. 线性基中的数线性无关（任意子集异或和不为0）
3. 线性基的大小是最小的

**核心应用**：求集合的最大异或和、第K小异或和、判断某个数能否被表示等。

## 板子1：高斯消元构造

### 问题

给定 n 个整数（数字可能重复），求在这些数中选取任意个，使得他们的异或和最大。

### 性质

用高斯消元得到的线性基具有**最简阶梯形**：
- 每一行的最高有效位（MSB）互不相同，且严格右移
- 从上到下单调递减

```
0...01....
0...001....
0...0001...
```

- `a[0]` 保存最高位的基向量（最高有效位最靠左）
- `a[k-1]` 保存最低位的基向量（最高有效位最靠右）
- `a[0]` 的最高位 ≥ `a[k-1]` 的最高位

### 代码实现

```cpp
#include<bits/stdc++.h>
using namespace std;
#define int long long
#define endl '\n'

int a[55];      // 下标从0开始
int n, k;       // k: 基向量个数

void Linear_base() {
    k = 0;
    // 1. 把第i位是1的数换上去
    // 2. 把其他第i位全部消成0
    for (int i = 62; i >= 0; i--) {
        for (int j = k; j < n; j++)
            if (a[j] >> i & 1) {
                swap(a[j], a[k]);
                break;
            }
        if ((a[k] >> i & 1) == 0) continue;
        
        for (int j = 0; j < n; j++)
            if (j != k && (a[j] >> i & 1))
                a[j] ^= a[k];
        
        k++;
        if (k == n) break;
    }
}

signed main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];
    
    Linear_base();
    
    int ans = 0;
    for (int i = 0; i < k; i++) ans ^= a[i];
    
    cout << ans;
    return 0;
}
```

## 板子2：贪心法构造

### 特点

- 更简洁，代码量更少
- 适合在线插入
- 不保证阶梯形，但功能完整

### 代码实现

```cpp
#include<bits/stdc++.h>
using namespace std;
#define int long long
#define endl '\n'

int n, p[64];

void insert(int x) {  // 贪心法
    for (int i = 63; i >= 0; --i)
        if (x >> i & 1) {
            if (!p[i]) {
                p[i] = x;
                break;
            }
            x ^= p[i];
        }
}

signed main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    cin >> n;
    int x;
    for (int i = 0; i < n; i++) {
        cin >> x;
        insert(x);
    }
    
    int ans = 0;
    for (int i = 63; i >= 0; i--)
        ans = max(ans, ans ^ p[i]);
    
    cout << ans;
    return 0;
}
```

## 应用1：第K小异或和

### 问题

HDU 3949 XOR - 给定n个数，求第K小的异或和。

### 思路

1. 用高斯消元构造线性基
2. 如果 `k < n`（线性基不满秩），说明0可以被表示，需要 `k--`
3. 将k二进制展开，对应位为1则异或对应的基向量

### 代码实现

```cpp
void solve() {
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];
    
    Linear_base();
    
    int q;
    cin >> q;
    while (q--) {
        int x;
        cin >> x;  // 第x小
        
        if (k < n) x--;  // 0可以被表示
        
        if (x >= (1ll << k)) {
            cout << -1 << endl;
        } else {
            int ans = 0;
            for (int i = 0; i < k; i++)
                if (x >> i & 1)
                    ans ^= a[k - i - 1];
            cout << ans << endl;
        }
    }
}
```

## 应用2：贪心法构造线性基

### P4570 [BJWC2011] 元素

**题意**：每个元素有编号和魔力值，选择若干元素使得编号异或和最大，且魔力值之和最大。

**思路**：
1. 按魔力值从大到小排序
2. 贪心地尝试插入线性基
3. 能插入就插入（说明这个元素有用），不能插入就跳过

### P4301 [CQOI2013] 新Nim游戏

**题意**：Nim游戏变种，先手可以在游戏开始前移除若干堆，使得后手无论如何都无法获胜。

**思路**：
1. 先手要让剩余的堆无论怎么取都无法异或和为0
2. 这等价于让剩余的堆构成线性基
3. 贪心：从大到小尝试插入，能插入就保留，不能插入就移除

## 应用3：图上最大异或路径

### P4151 [WC2011] 最大XOR和路径

**题意**：给定无向图，求从1到n的路径中，边权异或和最大的路径。

**思路**：
1. 先用DFS求出1到n的任意一条路径，记录其异或和
2. 枚举所有环，将环的异或和插入线性基
3. 用这条路径的异或和与线性基求最大异或和

**关键观察**：任意两条1到n的路径，其异或和的差值等于某些环的异或和。

```cpp
void dfs(int u, node fa) {
    for (node v : g[u]) {
        if (v == fa) continue;
        
        if (cal[v.id] != -1) {
            // 发现环
            insert(cal[u] ^ cal[v.id] ^ v.val);
        } else {
            cal[v.id] = cal[u] ^ v.val;
            dfs(v.id, {u, v.val});
        }
    }
}

int main() {
    // ... 建图 ...
    cal[1] = 0;
    dfs(1, {-1, -1});
    
    int ans = cal[n];
    for (int i = 63; i >= 0; i--)
        ans = max(ans, ans ^ p[i]);
    
    cout << ans;
}
```

## 应用4：前缀线性基

### 板子4：序列区间异或和

**1100F - Ivan and Burgers** - 给定序列，多次询问区间内选若干数的最大异或和。

**思路**：
1. 预处理前缀线性基 `pre[i]` 表示 `[1, i]` 的线性基
2. 查询 `[l, r]` 时，用 `pre[r]` 中所有数的最高位 ≥ l 的基向量

### 板子5：树上区间异或和

**P3292 [SCOI2016] 幸运数字** - 树上路径查询，求路径上选若干数的最大异或和。

**思路**：
1. 树链剖分 + 线性基
2. 或者倍增 + 线性基合并

## 应用6：实数线性基

### P3265 [JLOI2015] 装备购买

**问题**：给定n个向量（实数），每个向量有代价，选择若干向量使得线性无关，且代价最小。

**思路**：
1. 按代价从小到大排序
2. 贪心地尝试插入线性基（高斯消元）
3. 能插入就插入，不能插入就跳过

## 复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 高斯消元构造 | O(n × log²V) | O(n × log V) |
| 贪心法构造 | O(n × log V) | O(log V) |
| 查询最大异或和 | O(log V) | - |
| 查询第K小 | O(log V) | - |

其中 V 是值域。

## 常见技巧

### 1. 判断某个数能否被表示

```cpp
bool can_represent(int x) {
    for (int i = 63; i >= 0; i--)
        if (x >> i & 1) {
            if (!p[i]) return false;
            x ^= p[i];
        }
    return true;
}
```

### 2. 线性基合并

```cpp
void merge(int p1[], int p2[]) {
    for (int i = 63; i >= 0; i--)
        if (p2[i])
            insert(p2[i]);
}
```

### 3. 记录线性基的组成

使用位掩码记录每个基向量由哪些原始数组成（见牛客寒假营3 I题）。

## 总结

线性基是处理异或问题的核心工具：
1. **高斯消元法**：适合离线、需要阶梯形的场景
2. **贪心法**：适合在线、代码简洁的场景
3. **前缀线性基**：处理区间查询
4. **图上应用**：结合DFS处理环

掌握线性基后，大部分异或问题都能迎刃而解。
