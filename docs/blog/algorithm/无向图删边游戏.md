## 树的删边游戏
	- **问题：**有如下规则：
	  >给出一个有n个节点的有根树。
	  游戏者轮流从树中删去边，删去一条边后，不与根节点相连的部分将被移走。
	  无法操作者输
	- ## **结论：**叶子节点的SG值为0，中间节点的SG值为它所有子节点的SG值加1后的异或和。
	- ### 详细分析（思路1）：
		- 把“整棵树”当成一个**组合游戏**，我们需要给出其 **Grundy 数（SG 值）** 的数学表达式并证明。
		- ### 1. 约定根与方向
			- 为避免“哪一半被拿走”的歧义，**人为任选一个根**（通常把输入给出的 1 号点当成根）。  
			  于是每条边 `(u, parent[u])` 把整棵树切成子树 `T(u)`（以 `u` 为根的子树）
			- 其余部分  ：删除该边后，**只保留包含根的那部分**；`T(u)` 被丢掉。
			- 因此，整棵树等价于**“根的所有子树游戏”的并**，而每条子树游戏又独立递归。
		- ### 2. 子游戏的等价物
			- 对于以 `u` 为根的子树，把它看成一个独立游戏后，其 SG 值记为 `sg(u)`。  
			  关键观察：
			- 根 `u` 有 `k` 个儿子 `v1, v2, …, vk`
			- 砍掉边 `(u, vi)` 后，**只剩**以 `vi` 为根的那棵子树；  
			  这一步相当于把原游戏分裂成**一个子游戏 `sg(vi)`**。
			- 但注意：**砍掉这条边本身**也消耗了一次操作；  
			  在组合游戏理论中，这等价于给该子游戏**额外再添加一条“一阶”边**（即把 `sg(vi)` 整体“上移” 1 阶）。
			  于是得到递推式
			  ```
			  sg(u) = mex{ sg(vi) + 1 | vi 为 u 的儿子 }          (1)
			  ```
			  解释：
			- 每个可行动作对应把游戏变成 `(vi 子树的 SG) + 1`。
			- mex（最小非负整数）给出当前状态的 Grundy 值。
			- 边界：  
			  叶节点（无儿子）→ 无合法边可删 → SG = 0。
		- ### 3. 整棵树的 SG 值
			- 把整棵输入树以 1 为根跑一遍后，  
			  ```
			  SG(Tree) = sg(1)
			  ```
		- ### 4. 代码实现（O(n) DFS）
		  
		  ```cpp
		  vector<int> G[N];
		  int sg[N];
		  void dfs(int u,int fa) {
		    int g=0;
		    for(int v:G[u])
		        if(v!=fa){
		            dfs(v,u);
		            g^=(sg[v]+1);   // 公式 (1)
		        }
		    sg[u]=g;
		  }
		  int main(){
		    int n;
		    // 建树
		    dfs(1, 0);
		    puts(sg[1] ? "Alice" : "Bob");
		  }
		  ```
		- ### 5. 性质与结论
		  
		  | 性质 | 内容 |
		  |---|---|
		  | **对称性** | 任选根不改变整棵树的 SG，但递归式必须按“根-子树”方向写。 |
		  | **奇偶关键** | 对一条链：长度为 `L` 时 SG = L & 1；公式 (1) 退化验证正确。 |
		  | **森林** | 多棵树组成森林时，总 SG 为各树 SG 的异或和（普通 Nim 和）。 |
		  | **推广** | 无向图含环时，需先用 **Fusion Principle**（奇环缩成带权边，偶环直接缩点）把图压成树，再套同样公式。 |

- ## 无向图删边游戏
	- **问题：**有如下规则：
		- 一个无向联通图，有一个点作为图的根
		- 游戏者轮流从图中删去边，删去一条边后，不与根相连的部分将被移走。
		- 无法操作者输
	- ## Fusion Principle定理
	  
	  > 对无向图做如下改动：将图中任意一个==偶环缩成一个新点==，任意一个==奇环缩成一个新点加一个新边==；
	  所有连到原先环上的边全部与新点相连，这样的改动不会影响图的SG值。
		- 因此可以将任意无向图改成树形结构。
		- ### 定容显式结论：
		  >奇环的SG值为1
		  偶环的SG值为0
		- ### 定理证明！
			- 链的SG值即为长度，$sg(x)=x$
			- 奇环去掉任意条边后，剩下左右两侧的链长度同奇偶，其sg值即为所有情况的mex值
				- 1.同奇偶两数异或值一定为偶数
				- 2.当从中间对称点切开后两链长度一样，sg值为0
				- 故奇环的SG值为1
			- 偶环去掉任意条边后，剩下左右两侧的链长度一奇一偶，其sg值即为所有情况的mex值
				- 1.一奇一偶两数异或值一定为奇数
				- 故偶环的SG值一定为0
	- ### [难点！]环有重边情况：
		- 对于重边的处理是，==如果两点间有偶数条边，则当作偶数环处理，将其化为一个点==，否则保留。
	- ### 模板代码：
	  ```C++
	  #include <iostream>
	  #include <cstdio>
	  #include <string>
	  #include <string.h>
	  #include <map>
	  #include <vector>
	  #include <cstdlib>
	  #include <cmath>
	  #include <algorithm>
	  #include <cmath>
	  #include <queue>
	  #include <set>
	  #include <stack>
	  using namespace std;
	  vector<int>edge[105];
	  int mat[105][105];
	  int low[105], dfa[105];
	  int s[105], top;
	  bool instack[105];
	  bool vis[105];
	  void Tarjan(int u, int pre, int depth){
	      low[u] = dfa[u] = depth;
	      s[top++] = u;
	      instack[u] = true;
	      for(int i = 0; i < edge[u].size(); ++i){
	          int v = edge[u][i];
	          if(v == pre && mat[u][v]>1){
	              if(mat[u][v]%2==0)
	                  vis[u] = true;
	              continue;
	          }
	          if(!dfa[v]){
	              Tarjan(v, u, depth+1);
	              low[u] = min(low[u], low[v]);
	          }
	          else if(v!=pre && instack[v])
	              low[u] = min(low[u], dfa[v]);
	      }
	      if(dfa[u] == low[u]){
	          int cnt = 1;
	          top--;
	          while(s[top]!=u){
	              vis[s[top--]] = true;
	              cnt++;
	          }
	          if(cnt && (cnt&1))
	              vis[s[top+1]] = false;
	      }
	  }
	  int get_sg(int u, int pre){
	      int ret = 0;
	      for(int i =0; i < edge[u].size(); ++i){
	          int v = edge[u][i];
	          if(!vis[v] && v!=pre)
	              ret ^= (1+get_sg(v, u));
	      }
	      return ret;
	  }
	  int main(){
	      int k, n, m;
	      while(scanf("%d", &k)!=EOF){
	          int ret = 0;
	          while(k--){
	              scanf("%d%d", &n, &m);
	              for(int i =1; i <= n; ++i)
	                  edge[i].clear();
	              memset(mat, 0, sizeof(mat));
	              memset(low, 0, sizeof(low));
	              memset(dfa, 0, sizeof(dfa));
	              memset(instack, false, sizeof(instack));
	              memset(vis, false, sizeof(vis));
	              top = 0;
	              while(m--)
	              {
	                  int u, v;
	                  scanf("%d%d", &u, &v);
	                  mat[u][v]++;
	                  mat[v][u]++;
	                  edge[u].push_back(v);
	                  edge[v].push_back(u);
	              }
	              Tarjan(1, -1, 1);
	              ret ^= get_sg(1, -1);
	          }
	          puts(ret?"Sally":"Harry");
	      }
	      return 0;
	  }
	  ```
