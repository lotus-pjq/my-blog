- 题目- # P1337 [JSOI2004] 平衡点 / 吊打XXX
## 题目描述
如图，有 $n$ 个重物，每个重物系在一条足够长的绳子上。
每条绳子自上而下穿过桌面上的洞，然后系在一起。图中 $x$ 处就是公共的绳结。假设绳子是完全弹性的（即不会造成能量损失），桌子足够高（重物不会垂到地上），且忽略所有的摩擦，求绳结 $x$ 最终平衡于何处。
**注意**：桌面上的洞都比绳结 $x$ 小得多，所以即使某个重物特别重，绳结 $x$ 也不可能穿过桌面上的洞掉下来，最多是卡在某个洞口处。

## 输入格式
文件的第一行为一个正整数 $n$（$1\le n\le 1000$），表示重物和洞的数目。
接下来的 $n$ 行，每行是 $3$ 个整数 $x_i, y_i, w_i$，分别表示第 $i$ 个洞的坐标以及第 $i$ 个重物的重量。（$-10000\le x_i,y_i\le10000, 0三分
- 故三分套三分
- Tip：至于精度，eps需要1e-9，并且要开long double
- 代码
```C++
#include
using namespace std;
#define int long long
#define double long double 
#define endl '\n'
const int N=1e3+5;
const int INF=1e4+5;
const double eps=1e-9;
int n;
double ly,ry;
struct point{
double x,y,w;
point(double x, double y):x(x),y(y),w(w){}
point():x(0),y(0),w(0){}
}p[N];
double dis(double x,double y,point a){return sqrtl((a.x-x)*(a.x-x)+(a.y-y)*(a.y-y));}
double cal(double x,double y){
double res=0;
for(int i=1;i>n;
for(int i=1;i>p[i].x>>p[i].y>>p[i].w;
double lx=-INF,rx=INF;
while(fabs(rx-lx)>eps){
double dx=(rx-lx)/3,x1=lx+dx,x2=x1+dx;

ly=-INF,ry=INF;
while(fabs(ry-ly)>eps){
double dy=(ry-ly)/3,y1=ly+dy,y2=y1+dy;
double cal1=cal(x1,y1),cal2=cal(x1,y2);
if(cal1>cal2) ly=y1;
else ry=y2;
}
double calx1=cal(x1,ly);

ly=-INF,ry=INF;
while(fabs(ry-ly)>eps){
double dy=(ry-ly)/3,y1=ly+dy,y2=y1+dy;
double cal1=cal(x2,y1),cal2=cal(x2,y2);
if(cal1>cal2) ly=y1;
else ry=y2;
}
double calx2=cal(x2,ly);

if(calx1>calx2) lx=x1;
else rx=x2;
}
printf("%.3Lf %.3Lf",lx,ly);
return 0;
}
```
