# 求和的优化计算

我将帮你解决这个需要双重求和的问题，并通过数学变换将其转化为更高效的计算方式。
- ## 问题描述
  
  我们需要计算：
  \[ \sum_{1 \leq i \leq j \leq n} (a_i + a_j) \cdot (w_j - w_i) \]
- ## 解法分析
- ### 数学变形
  我们可以展开这个表达式：
  \[ (a_i + a_j)(w_j - w_i) = a_i w_j + a_j w_j - a_i w_i - a_j w_i \]
  因此，原式可以拆分为：
  \[ \sum_{i \leq j} (a_i w_j + a_j w_j - a_i w_i - a_j w_i) \]
  这可以进一步分解为四个求和：
  1. \( \sum_{i \leq j} a_i w_j \)
  2. \( \sum_{i \leq j} a_j w_j \)
  3. \( -\sum_{i \leq j} a_i w_i \) 
  4. \( -\sum_{i \leq j} a_j w_i \)
- ### 单独计算每一部分
  **第一部分和第二部分**：
  \[ \sum_{i \leq j} a_i w_j = \sum_{j=1}^n w_j \sum_{i=1}^j a_i \]
  \[ \sum_{i \leq j} a_j w_j = \sum_{j=1}^n a_j w_j \cdot j \]
  **第三部分**：
  \[ -\sum_{i \leq j} a_i w_i = -\sum_{i=1}^n a_i w_i (n - i + 1) \]
  **第四部分**：
  \[ -\sum_{i \leq j} a_j w_i = -\sum_{i=1}^n w_i \sum_{j=i}^n a_j \]
- ### 最终表达式
  将以上部分合并：
  \[ \text{结果} = \sum_{j=1}^n w_j S_{a,j} + \sum_{j=1}^n j a_j w_j - \sum_{i=1}^n (n-i+1) a_i w_i - \sum_{i=1}^n w_i (S_{a,n} - S_{a,i-1}) \]
  其中 \( S_{a,k} \) 表示 a 数组的前缀和。
- ### 进一步化简
  可以进行进一步代数化简，但实际实现时可以直接计算上述四个部分的和。
- ## 复杂度分析
- 计算前缀和：O(n)
- 计算四个部分的和：每个部分需要 O(n) 时间
- 总时间复杂度：O(n)
- 空间复杂度：O(n)（用于存储前缀和）
  
  这种方法将原始的 O(n²) 双重求和问题优化为 O(n) 的线性复杂度，大大提高了效率。
