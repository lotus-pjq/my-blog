## 🧩 1. 题目抽象
给定多项式  
F(x) = f₀ + f₁x + f₂x² + … + fₙ₋₁xⁿ⁻¹  (f₀ ≠ 0)  
求多项式 G(x) 使得  
F(x) · G(x) ≡ 1 (mod xⁿ)  
即 G(x) 是 F(x) 在模 xⁿ 意义下的 **逆元**。  
输出 G(x) 的前 n 个系数（低次 → 高次）。
---
## ⚙️ 2. 算法总览
  使用 **牛顿迭代** + **NTT**（数论变换）
- 牛顿迭代：把求逆元转化为若干次 **多项式乘法**
- NTT：在 O(n log n) 时间内完成乘法
  ---
