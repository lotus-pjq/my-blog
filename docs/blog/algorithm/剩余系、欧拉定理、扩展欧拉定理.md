- 知识点- 
- 试除法求欧拉函数 O(√n)

```C++
int phi(int x){
    int res=x;
    for(int i=2;i*i1) res=res-res/x;
    return res;
}
```
- 扩展欧拉定理

```C++
int ex_eular(int a,string b,int c){//a**b(modc)
    int ans=0,phic=phi(c);
    bool flag=0;//标记b是否大于等于φ(c)
    for(int i=0;i=phic){ans%=phic;flag=true;}
    }
    if(flag) ans+=phic;  
    return ksm(a,ans,c);
}
```
- 板子题- 题目- # P5091 【模板】扩展欧拉定理
  ## 题目描述
给你三个正整数，$a,m,b$，你需要求：$a^b \bmod m$
  ## 输入格式
一行三个整数，$a,m,b$
  ## 输出格式
一个整数表示答案
  ## 输入输出样例 1
  ### 输入 1
```
2 7 4
```
  ### 输出 1
```
2
```
  ## 输入输出样例 2
  ### 输入 2

```
998244353 12345 98765472103312450233333333333
```
  ### 输出 2
```
5333
```
  ## 说明/提示
注意输入格式，$a,m,b$ 依次代表的是底数、模数和次数   
【样例 $1$ 解释】   
$2^4 \bmod 7 = 2$     
【数据范围】  
对于 $100\%$ 的数据，$1\le a \le 10^9$，$1\le b \le 10^{20000000}，1\le m \le 10^8$。
- 代码
```C++
#include
using namespace std;
#define int long long
#define endl '\n'
const int N=1e5+5;
int a,c;
string b;
int ksm(int a,int b,int c){
int ans=1;
while(b){
if(b&1) ans=(ans*a)%c;
b>>=1;a=(a*a)%c;
}return ans;
}
int phi(int x){
int res=x;
for(int i=2;i*i1) res=res-res/x;
return res;
}
int ex_eular(int a,string b,int c){//a**b(modc)
int ans=0,phic=phi(c);
bool flag=0;//标记b是否大于等于φ(c)
for(int i=0;i=phic){ans%=phic;flag=true;}
}
if(flag) ans+=phic;  
return ksm(a,ans,c);
}
signed main(){
ios::sync_with_stdio(false);
cin.tie(nullptr);
cin>>a>>c>>b;
cout<<ex_eular(a,b,c);
}
```
