	- 题目
		- # P3807 【模板】卢卡斯定理/Lucas 定理
		- ## 题目背景
		  这是一道模板题。
		- ## 题目描述
		  给定整数 $n, m, p$ 的值，求出 $C_{n + m}^n \bmod p$ 的值。
		  输入数据保证 $p$ 为质数。
		  注: $C$ 表示组合数。
		- ## 输入格式
		  **本题有多组数据**。
		  第一行一个整数 $T$，表示数据组数。
		  对于每组数据: 
		  一行，三个整数 $n, m, p$。
		- ## 输出格式
		  对于每组数据，输出一行，一个整数，表示所求的值。
		- ## 输入输出样例 1
		- ### 输入 1
		  ```
		  2
		  1 2 5
		  2 1 5
		  ```
		- ### 输出 1
		  ```
		  3
		  3
		  ```
		- ## 说明/提示
		  对于 $100\%$ 的数据，$1 \leq n, m, p \leq 10^5$，$1 \leq T \leq 10$。
	- 代码
		- ```C++
		  #include<bits/stdc++.h>
		  using namespace std;
		  #define int long long
		  const int mod=1e9+7;
		  const int N=1e5+5;
		  int n,m,a[N],p;
		  int tt[N],intt[N];
		  void init_inff(int mod){
		      tt[0]=intt[0]=1;
		      tt[1]=intt[1]=1;
		      for(int i=2;i<N;i++){
		          tt[i]=i;
		          intt[i]=(mod-mod/i)*intt[mod%i]%mod;
		      }
		      for(int i=2;i<N;i++){
		          tt[i]=tt[i-1]*tt[i]%mod;
		          intt[i]=intt[i-1]*intt[i]%mod;
		      }
		  }
		  int C(int n,int m,int mo){
		      if(n<0||m<0||m>n) return 0;
		      return tt[n]*intt[m]%mo*intt[n-m]%mo;
		  }
		  int Lucas(int a,int b,int mo){
		      if(b==0) return 1;
		      return C(a%mo,b%mo,mo)*Lucas(a/mo,b/mo,mo)%mo;
		  }
		  void solve(){
		      cin>>n>>m>>p;
		      init_inff(p);
		      cout<<Lucas(n+m,m,p)<<endl;
		  }
		  signed main(){
		      ios::sync_with_stdio(false);
		      cin.tie(nullptr);
		      int t;cin>>t;
		      while(t--) solve();
		      return 0;
		  }
		  ```
- 应用1：组合数模2的性质
	- 根据卢卡斯定理，组合数 \(\binom{n}{m} \equiv \binom{\lfloor \frac{n}{2} \rfloor}{\lfloor \frac{m}{2} \rfloor} \times \binom{n \mod 2}{m \mod 2} \pmod{2}\)。
	- 于是，设 \(n\) 的二进制分解为 \(n = \sum a_i 2^i\)，\(m\) 的二进制分解为 \(n = \sum b_i 2^i\)。
	- 于是，\(\binom{n}{m} \equiv \sum \binom{a_i}{b_i} \pmod{2}\)。
	- 考虑，\(\binom{0}{0} = 1\)，\(\binom{1}{0} = 0\)，\(\binom{0}{1} = 1\)，\(\binom{1}{1} = 1\)，
	- 于是 \(\binom{n}{m} \equiv 1 \pmod{2}\) 当且仅当不能出现 \(a_i = 0\)，\(b_i = 1\)，于是即为 \(n\) 在二进制下包含 \(m\)。
