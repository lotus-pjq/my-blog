- E.神秘异或操作
	- 
	- 先不考虑这个特殊的count奇偶问题，考虑普通操作后sigema_N_N_Ai^Aj.
	- ### 对于异或这种位运算的操作，不妨从每一位的贡献开始考虑，最后只需要每位贡献累计即可（这个过程只有logN复杂度）。
	- **<按位考虑>**
	- 普通操作的值：当前k(0开始)数位上的数，这n个数一共有cnt1个1，cnt0个0，那么求和贡献为(1<<k)*cnt1*cnt0
	- 那么回到本题这个特殊操作，
	- 多了一个限制条件，若两个数x和y该位k在异或后想要把1保留下来，需要x^y的低k-1位中有偶数个1。
	- **<等价结论>**：设x的低k-1位有cntx个1，同理cnty，当前仅当cntx和cnty的奇偶性相同的时候低位会有偶数个1。
	- 结论证明：当x和y低k-1位相同位置的1有pub个，则k-1位异或后的1的个数为(cntx-pub)+(cnty-pub)=(cntx+cnty)-2*pub，只要cntx和cnty的奇偶性相同则能保证有偶数个1。
	- ```C++
	  #include<bits/stdc++.h>
	  using namespace std;
	  #define int long long
	  #define endl '\n'
	  const int N=2e5+5;
	  int a[N];
	  int pre1[N];
	  signed main(){
	  	ios::sync_with_stdio(false);
	  	cin.tie(nullptr);
	  	int n,ans=0;cin>>n;
	  	for(int i=1;i<=n;i++) cin>>a[i];
	  	for(int k=0;k<=30;k++){
	  		int p00=0,p10=0,p01=0,p11=0;
	  		for(int i=1;i<=n;i++){
	  			int x=(a[i]>>k)&1;
	  			if(x){
	  				if(pre1[i]&1) p11++;
	  				else p10++;
	  			}else{
	  				if(pre1[i]&1) p01++;
	  				else p00++;
	  			}
	  			pre1[i]+=x;
	  		}
	  		ans+=(1ll<<k)*(p11*p01+p10*p00);
	  	}
	  	cout<<ans<<endl;
	  	return 0;
	  }
	  ```
