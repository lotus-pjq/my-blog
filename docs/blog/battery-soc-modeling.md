# 连续时间电池 SOC 建模笔记

最近整理电池 SOC 建模时，我更关心的不是“模型越复杂越好”，而是模型能否清楚说明参数含义、适用区间和实验验证方式。

## 当前关心的三个部分

### 1. 状态方程怎么写

二阶 RC 模型适合描述动态响应，KiBaM 更方便表达可用容量与束缚容量之间的交换。把两者放在同一份笔记里，有助于比较不同建模思路的解释力。

### 2. 温度项怎么接入

如果温度只作为一个后验修正项，表达会很弱。我更倾向于把温度影响拆到参数层，而不是只在最后乘一个经验系数。

### 3. 如何验证

模型写出来之后，至少要回答三个问题：

- 参数怎么拟合
- 实验数据怎么清洗
- 误差怎么报告
